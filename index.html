HTML Cheat Sheet 🚀

Tu guía rápida para los elementos HTML esenciales.

Estructura Básica del Documento

<!DOCTYPE html>

Funcionalidad: Es la declaración de tipo de documento. Debe ser la primera línea del HTML. Indica al navegador que el documento es HTML5.

<!DOCTYPE html>
<html>

Funcionalidad: Es el elemento raíz de cualquier página web HTML. Contiene todo el contenido de la página.

<html lang="es"></html>
<head>

Funcionalidad: Contiene metadatos sobre el documento (información no visible en la página, como el título, enlaces a CSS, scripts, etc.).

<head>\n  <meta charset="UTF-8">\n  <title>Mi Página</title>\n</head>
<body>

Funcionalidad: Contiene todo el contenido visible de la página web (texto, imágenes, videos, enlaces, etc.).

<body>\n  <h1>Hola Mundo</h1>\n</body>
<title>

Funcionalidad: Define el título de la página que aparece en la pestaña del navegador o en los resultados de búsqueda. Va dentro de <head>.

<title>Título de mi Pestaña</title>

Estructura Semántica y Layout

<header>

Funcionalidad: Representa contenido introductorio o un grupo de enlaces de navegación. Usualmente contiene el logotipo, el título del sitio, y la navegación principal.

<header>\n  <img src="logo.png" alt="Logo">\n  <h1>Mi Sitio</h1>\n</header>
<nav>

Funcionalidad: Define un bloque de enlaces de navegación. Se usa para la navegación principal del sitio.

<nav>\n  <ul>\n    <li><a href="#">Inicio</a></li>\n    <li><a href="#">Acerca de</a></li>\n  </ul>\n</nav>
<main>

Funcionalidad: Representa el contenido dominante del <body>. Solo debe haber un elemento <main> por documento.

<main>\n  <h2>Contenido Principal</h2>\n  <p>Aquí va el texto relevante.</p>\n</main>
<aside>

Funcionalidad: Representa una sección de contenido relacionada indirectamente con el contenido principal. Se usa para barras laterales, bloques de anuncios, etc.

<aside>\n  <h3>Artículos Relacionados</h3>\n  <ul>...</ul>\n</aside>
<footer>

Funcionalidad: Contiene información sobre el documento o su sección más cercana (autor, derechos de autor, enlaces relacionados, etc.).

<footer>\n  <p>&copy; 2025 Mi Empresa</p>\n</footer>
<div>

Funcionalidad: Un contenedor genérico de nivel de bloque que no tiene significado semántico por sí mismo. Se usa para agrupar elementos y aplicar estilos CSS.

<div class="card">\n  <h3>Título</h3>\n  <p>Contenido.</p>\n</div>
<span>

Funcionalidad: Un contenedor genérico en línea (inline) que no tiene significado semántico. Se usa para aplicar estilos CSS a una parte del texto u otros elementos en línea.

<p>Este es un texto con una <span style="color: blue;">parte azul</span>.</p>

Texto y Encabezados

<p>

Funcionalidad: Define un párrafo de texto. Es el elemento básico para bloques de texto en HTML.

<p>Este es un párrafo de ejemplo.</p>
<h1> a <h6>

Funcionalidad: Definen encabezados HTML. <h1> es el más importante (título principal) y <h6> el menos importante (subtítulo de menor jerarquía).

<h1>Título Principal</h1>\n<h3>Subtítulo</h3>

Enlaces y Listas

<a>

Funcionalidad: Crea un hiperenlace (link) a otras páginas web, archivos, ubicaciones dentro de la misma página, o direcciones de correo electrónico.

<a href="https://www.google.com" target="_blank">Ir a Google</a>
<ol>, <ul>, <dl>

Funcionalidad:

  • <ol> (Ordered List): Lista ordenada (numerada).
  • <ul> (Unordered List): Lista no ordenada (con viñetas).
  • <dl> (Description List): Lista de descripción (pares término-descripción).

<ol>\n  <li>Paso 1</li>\n</ol>\n\n<ul>\n  <li>Elemento</li>\n</ul>\n\n<dl>\n  <dt>Término</dt>\n  <dd>Descripción</dd>\n</dl>
<li>, <dt>, <dd>

Funcionalidad: Elementos de lista.

  • <li> (List Item): Elemento de una lista (<ol> o <ul>).
  • <dt> (Definition Term): Término a definir en una lista de descripción (<dl>).
  • <dd> (Definition Description): Descripción del término en una lista de descripción (<dl>).

<ul>\n  <li>Elemento de lista</li>\n</ul>\n\n<dl>\n  <dt>HTML</dt>\n  <dd>Lenguaje de Marcado de Hipertexto</dd>\n</dl>

Tablas

<table>, <tr>, <td>, <th>

Funcionalidad:

  • <table>: Contenedor para toda la tabla.
  • <tr> (Table Row): Fila de la tabla.
  • <td> (Table Data): Celda de datos estándar.
  • <th> (Table Header): Celda de encabezado (usualmente negrita y centrada).

<table>\n  <tr>\n    <th>Encabezado</th>\n    <th>Encabezado 2</th>\n  </tr>\n  <tr>\n    <td>Dato 1</td>\n    <td>Dato 2</td>\n  </tr>\n</table>
<thead>, <tbody>, <tfoot>

Funcionalidad: Agrupan semánticamente las filas de una tabla.

  • <thead>: Contiene el contenido de encabezado (usualmente <th>).
  • <tbody>: Contiene el contenido principal del cuerpo de la tabla.
  • <tfoot>: Contiene el contenido del pie de tabla (sumarios, totales).

<table>\n  <thead><tr><th>Producto</th></tr></thead>\n  <tbody><tr><td>Manzana</td></tr></tbody>\n  <tfoot><tr><td>Total</td></tr></tfoot>\n</table>

Contenido Multimedia

<img>, <audio>, <video>, <source>

Funcionalidad:

  • <img>: Incrusta una imagen en el documento. Atributos clave: src (ruta), alt (texto alternativo).
  • <audio>: Incrusta contenido de audio. Atributos: controls, autoplay, loop.
  • <video>: Incrusta un reproductor de video. Atributos: controls, width, height, poster.
  • <source>: Especifica múltiples fuentes de medios para <audio> o <video>, permitiendo al navegador elegir la más compatible.

<img src="imagen.jpg" alt="Descripción">\n\n<audio controls>\n  <source src="audio.mp3" type="audio/mpeg">\n</audio>\n\n<video controls width="300">\n  <source src="video.mp4" type="video/mp4">\n</video>

Formularios

<form>

Funcionalidad: Crea una sección que contiene controles interactivos para que el usuario envíe información (ej. campos de texto, botones, casillas de verificación).

<form action="/submit" method="post">\n  <!-- Elementos del formulario -->\n</form>
<input>

Funcionalidad: Crea un control interactivo para formularios. El atributo type define su comportamiento (text, password, checkbox, radio, submit, etc.).

<input type="text" name="nombre" placeholder="Tu nombre">\n<input type="password" name="pass">
<select>, <option>

Funcionalidad:

  • <select>: Crea un control que proporciona un menú de opciones.
  • <option>: Define una opción dentro de un elemento <select> o <datalist>.

<select name="fruta">\n  <option value="manzana">Manzana</option>\n  <option value="pera">Pera</option>\n</select>
<textarea>

Funcionalidad: Crea un control de edición de texto multi-línea. Útil para comentarios largos o descripciones.

<textarea name="mensaje" rows="4" cols="30">Escribe aquí...</textarea>
<label>

Funcionalidad: Asocia un texto con un elemento de formulario (ej. <input>). Mejora la accesibilidad: al hacer clic en la etiqueta, el foco se va al control asociado.

<label for="email">Email:</label>\n<input type="email" id="email">
